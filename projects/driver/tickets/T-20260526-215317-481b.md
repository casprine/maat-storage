## Description

Maintain the web console in apps/web as a quiet operational UI for household document management, using Vite, React, TypeScript, Tailwind, OXC, lucide icons, shadcn-style components, and generated SDK-backed API helpers.

## Acceptance

- The UI supports setup/login, document list, upload, search, detail with preview/text/events, settings, tags, collections, retry, and sign out.
- Frontend API calls use packages/sdk generated functions or a thin local wrapper over them, not independent endpoint contracts.
- Typecheck and OXC lint pass for frontend changes.
- Responsive layout avoids overlapping text and keeps document workflows efficient.
