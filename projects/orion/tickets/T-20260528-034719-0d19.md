## Description

Add Console controls for the Agent token lifecycle design: show non-secret token status on Agent detail, provide admin rotate revoke and reissue actions, show one-time replacement tokens safely, and guide operators that rotation preserves identity while reconfigure can create a new identity.

## Acceptance

- Console uses generated SDK types for token status and lifecycle actions; Agent detail shows active revoked rotated and version metadata without secrets; rotate and reissue display the new token once with no persistence; revoke requires confirmation and reason entry; build and browser smoke verify no token leaks in ordinary Agent views.
