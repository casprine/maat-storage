## Description

Turn common CLI failures and setup doctor checks into specific desktop states instead of generic error dialogs.

## Acceptance

- Missing CLI, missing setup, invalid storage, Git credential failures, index failures, and merge conflicts each have distinct UI handling.
- CLI stderr and JSON details remain available for troubleshooting.
- A successful Markdown write followed by an index failure is shown as durable state with a rebuild option, not as a failed write retry.
