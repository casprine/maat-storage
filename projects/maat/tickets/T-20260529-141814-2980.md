## Description

Make index refresh code write only under the already resolved storage root and fail loudly when storage resolution is absent or ambiguous.

## Acceptance

- WriteIndex and SQLite rebuild callers receive an explicit resolved storage path.
- Index refresh never independently falls back to cwd.
- Failures report that indexes are rebuildable and identify the intended storage path when available.
