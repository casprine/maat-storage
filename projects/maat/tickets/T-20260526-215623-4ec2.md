## Description

Record the binary name, install directory, and target path during successful install/update so future bare uninstall commands remove the same binary.

## Acceptance

- Successful update or install script runs persist the binary location in local Maat config.
- Bare maat uninstall uses the remembered location when --install-dir and --binary-name are omitted, and clears stale install metadata after removal.
