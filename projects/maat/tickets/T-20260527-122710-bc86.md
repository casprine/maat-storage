## Description

Implement the update model for a Tauri app that ships with a bundled CLI while still allowing the app to know and verify the CLI it is using.

## Acceptance

- App updates can replace the bundled CLI and refresh the app-private installed CLI.
- The app records and displays the active CLI path and version.
- Advanced CLI self-update behavior through maat update --json is either implemented or explicitly deferred.
