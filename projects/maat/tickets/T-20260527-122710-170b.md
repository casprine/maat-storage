## Description

Create the desktop setup flow for selecting, creating, or cloning a Git-backed Maat storage repository, then configure the CLI with maat setup and validate it with setup doctor.

## Acceptance

- A user can create a new local storage repo, select an existing repo, or enter a clone URL when clone support is enabled.
- The app runs maat setup --storage <path> --actor <name> --json after storage selection.
- The app runs maat setup doctor --storage <path> --fix --json and shows blocking checks, warnings, and safe fixes.
