## Description

Package a signed maat binary inside the macOS app and install or update an app-private copy under Application Support during first-run setup.

## Acceptance

- The app bundle includes the correct darwin maat binary for the target architecture.
- On first launch, the bundled binary can install or refresh ~/Library/Application Support/maat/bin/maat using maat update --source ... --json.
- The app records the installed CLI path and verifies it with maat version --json before using it.
